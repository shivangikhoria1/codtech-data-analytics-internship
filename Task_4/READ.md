# 🤖 Task 4: Sentiment Analysis using NLP

## 📌 Overview

This project performs sentiment analysis on textual data (movie reviews) using Natural Language Processing (NLP) techniques. The goal is to classify reviews as positive or negative.

## 📂 Dataset

* IMDB Movie Reviews Dataset
* Total Records: 50,000
* Balanced dataset with positive and negative sentiments

## 🛠️ Tools & Technologies

* Python
* Pandas
* NLTK
* Scikit-learn

## 🔍 Steps Performed

### 1. Data Preprocessing

* Converted text to lowercase
* Removed special characters
* Removed stopwords
* Applied stemming

### 2. Feature Extraction

* Used TF-IDF (Term Frequency-Inverse Document Frequency)

### 3. Model Building

* Logistic Regression model

### 4. Model Evaluation

* Accuracy Score
* Classification Report
* Confusion Matrix

* Model achieved approximately **XX% accuracy**

## 🧠 Key Insights

* TF-IDF effectively converts text into numerical format
* Logistic Regression works well for binary classification
* Balanced dataset improves performance

This project demonstrates how NLP techniques can be used to analyze textual data and classify sentiments effectively.
